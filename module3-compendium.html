Felix

To enable ds on felix (download jars from http://felix.apache.org/downloads.cgi#subprojects)
install http://mirror.nexcess.net/apache/felix/org.apache.felix.scr-2.0.10.jar

install http://mirror.nexcess.net/apache/felix/org.apache.felix.configadmin-1.8.14.jar

Equinox

To enable ds on equinox

install file:/<path to your equinox sdk>/plugins/org.eclipse.equinox.util_1.0.500.v20130404-1337.jar

install file:/<path to your equinox sdk>/plugins/org.eclipse.equinox.cm_1.1.200.v20160324-1850.jar

install file:/<path to your equinox sdk>/plugins/org.eclipse.osgi.services_3.5.100.v20160504-1419.jar

start these three along with declarative services
